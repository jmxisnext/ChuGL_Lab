# Assets

Place shared media files here:

- **textures/** — PNG / JPEG images used as textures
- **audio/** — audio samples (`.wav`, `.aiff`) referenced by patches
- **fonts/** — TrueType / OpenType fonts

> Large binary files (audio, video) are excluded from git via `.gitignore`.
> Consider [Git LFS](https://git-lfs.com/) if you need to track large assets.
